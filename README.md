# 🚀 JSCamp - Proyecto de Aprendizaje Personal

Este repositorio contiene mis resoluciones prácticas, apuntes y experimentos desarrollados paso a paso a lo largo del bootcamp intensivo de JavaScript y desarrollo full-stack **JSCamp**. 

El objetivo de este proyecto es consolidar mis conocimientos en el desarrollo web moderno, arquitectura de software, bases de datos y DevOps.

---

## 🎨 El Proyecto Práctico: InfoJobs Clone
A lo largo de este bootcamp, se construye una aplicación real y completa de forma incremental aplicando los conocimientos de cada módulo:
* **Diseño del Proyecto**: [Ver diseño en Stitch (Google)](https://stitch.withgoogle.com/projects/7508115667617706440)
* **Alcance**: Frontend dinámico, APIs REST escalables, bases de datos relacionales, automatización de pruebas, containerización con Docker y despliegues automáticos (CI/CD).

## 💻 Requisitos de Instalación

Antes de comenzar a explorar y ejecutar los diferentes módulos del proyecto, asegúrate de contar con el siguiente software instalado de manera local:

* **Node.js** (Versión 20 o superior)
* **Docker & Docker Desktop** (Indispensable para el módulo de contenedores e IA local)
* **Git** (Para control de versiones)
